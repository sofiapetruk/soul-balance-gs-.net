## Instruções para Executação da API
### 1. Pré-requisitos
  - Certifique-se de ter o .NET SDK 8.0 ou superior instalado em sua máquina.

### 2. Clonar o Repositório
  - git clone https://github.com/sofiapetruk/soul-balance-gs-.net.git

### 3. Navegar até a pasta do projeto
  - cd .\soulBalanceGs\

### 4. Rodar a aplicação
  - dotnet run

### 5. Acesso a API
  - http://localhost:5053/swagger/index.html
  - http://localhost:5053/swagger/
    
## Instruções para rodar os Testes Unitários
  1. cd TestProject
  2.  dotnet test
